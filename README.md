# Titan-Terminal
Bloomberg-style terminal for stock analysis with 100% local AI. Runs 6 LLMs (DeepSeek, Gemma, Phi-4, Qwen, LLaMA, GLM4) in parallel via Ollama. Features: financials, technicals, ownership, options, news sentiment, peer comparison, and multi-model investment verdicts. Your data never leaves your machine. Privacy-first investing.
